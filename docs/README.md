# Documentación del Proyecto Maps.eus

Este directorio contiene guías y documentación técnica para el desarrollo del ecosistema Maps.eus.

## Guías Disponibles

*   [Configuración de Google Auth](google_auth_setup.md): Pasos para habilitar el login con Gmail.
*   **Debug en Docker**: Configurado en `.vscode/launch.json` y `tasks.json`.

## Estructura de Desarrollo

*   `src/`: Código fuente de Django.
*   `docker/`: Configuraciones adicionales de Docker (si las hubiera).
*   `.vscode/`: Configuración del IDE para depuración y tareas.
